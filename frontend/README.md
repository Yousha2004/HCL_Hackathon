# HealMe

Health and Wellness tracker application frontend.

# Stack
Built with Next.js App router and shadcn.

## Getting Started

First, run the development server:

```bash
pnpm i
cp .env.example .env.local# copy environment variables from example file
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Inter](https://vercel.com/font), a new font family for Vercel.
